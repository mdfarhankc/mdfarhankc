## About

I build developer tooling, mostly the kind that deletes boilerplate. Most of it started
as something I needed myself and didn't want to write twice: an auth layer for FastAPI,
a project scaffold I kept copying between repos, a REST framework that makes Odoo feel
like FastAPI, a CLI for the Odoo chores I do every day.

Day to day I'm a Python developer working across FastAPI, Django and Odoo. Outside of
that I build cross-platform apps with Flutter and Next.js, and I'm currently exploring
machine learning and AI-driven automation.
